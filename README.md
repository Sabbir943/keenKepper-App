## 🌟 Project Overview

**KeenKeeper** is a modern friendship management web app that helps users maintain meaningful relationships. It allows users to track interactions, set contact goals, and stay connected with friends through an intuitive and responsive interface.

---


## 🛠️ Technologies Used

- ⚛️ React.js  
- 🔀 React Router DOM  
- 🎨 Tailwind CSS  
- 📊 Recharts  
- 🔔 React Toastify (or any toast library)  
- 📦 JSON (for data storage)  

---

## ✨ Key Features

### 👫 Friend Management
- Display friends using dynamic cards
- View detailed friend profiles
- Track contact status (overdue / almost due / on-track)

### ⚡ Interaction Tracking
- Log interactions (Call, Text, Video)
- Auto-generate timeline entries
- Show toast notifications on actions

### 📜 Timeline System
- View complete interaction history
- Filter interactions (Call / Text / Video)
- Dynamic updates from user actions
